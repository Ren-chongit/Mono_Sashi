# 物差しアプリ (Monosashi)

画面上に自由に動かせる縦横の定規を表示するアプリケーションです。
半透明で背面のアプリケーション（Excelなど）を透かして見ることができ、定規越しに操作も可能な「クリックスルー」機能を備えています。

## 特徴
- **最前面表示**: 常に他のウィンドウの上に表示されます。
- **クリックスルー**: 定規の線や目盛りの上をクリックしても、背面のウィンドウを操作できます。
- **半透明デザイン**: 画面を遮らず、作業の邪魔になりません。
- **目盛り付き**: ピクセル単位（目安）の目盛りと数値が表示されます。
- **0点位置**: 縦定規と横定規の交差する内側の角が「0」になっています。
- **キーボード操作**: 矢印キーで1px単位の微調整が可能です。

## 使い方
1. `Monosashi_v5.exe`（または最新のexe）を起動します。
2. 画面に十字の定規が表示されます。
3. **交差位置（透明な四角形）**をドラッグして、好きな位置に移動させます。
4. **矢印キー** (`↑` `↓` `←` `→`) で位置を微調整できます。
    - `Shift` + 矢印キーで大きく移動します。
5. 終了するには `Esc` キーを押してください。

## 配布用ファイル
ビルド（exe化）済みの実行ファイルは `Mono_Sashi` フォルダ内に生成されています。

- `Mono_Sashi/Monosashi_v5.exe`: 最新版（アイコン適用済み）
- `Mono_Sashi/使い方マニュアル.txt`: 初心者向け説明書

## 開発環境
- Python 3.11
- Tkinter (GUI)
- pywin32 (Windows API制御)
- PyInstaller (exe化)
- Pillow (画像処理)

## ビルド方法
開発環境でexeファイルを再作成する場合は、以下のコマンドを実行してください。
（アイコンファイル `icon.ico` が必要です）

```bash
uv run pyinstaller --onefile --noconsole --name Monosashi_v5 --clean --icon=icon.ico main.py
```
