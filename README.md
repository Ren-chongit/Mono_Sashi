# 物差しアプリ (Monosashi)

画面上に自由に動かせる縦横の定規（赤線）を表示するアプリケーションです。
交差位置にあるハンドルを使って移動させることができ、それ以外の部分は背面のアプリケーション（Excelなど）をそのまま操作できる「クリックスルー」機能を持っています。

## 機能概要
- **最前面表示**: 常に他のウィンドウの上に表示されます。
- **クリックスルー**: 線（交差部以外）はマウス操作を無視し、背面のウィンドウを操作できます。
- **移動**: 十字線の交差位置にある赤い四角（ハンドル）をドラッグすると、定規全体が移動します。
- **終了**: `Esc` キーを押すとアプリケーションが終了します。

## 使い方
1. `Monosashi.exe` を起動します。
2. 画面に赤い十字線が表示されます。
3. **交差位置の赤い四角**をドラッグして、好きな位置に移動させます。
4. 線以外の場所（線の上を含む）をクリックして、背面のウィンドウ（Excelなど）を操作します。
5. 終了するには `Esc` キーを押してください。

## 配布用ファイル
ビルド済みの実行ファイルは以下の場所にあります。このファイルをコピーして配布・使用できます。

`dist/Monosashi.exe`

## 開発環境
- Python 3.11
- Tkinter (GUI)
- pywin32 (Windows API制御)
- PyInstaller (exe化)

## ビルド方法
開発環境でexeファイルを再作成する場合は、以下のコマンドを実行してください。

```bash
uv run pyinstaller --onefile --noconsole --name Monosashi --clean main.py
```
